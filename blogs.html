What are the different ways to select an element in the DOM?

You can select elements in JavaScript using different methods, such as:

  • document.getElementById()
  • document.getElementsByClassName()
  • document.getElementsByTagName()
  • document.querySelector()
  • document.querySelectorAll()

What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML: Retrieves and sets HTML content, including tags.
  • innerText: Extracts only visible text, ignoring hidden elements.
  • textContent: Retrieves all text, including hidden content, without parsing HTML.

What is event delegation in the DOM?

Instead of attaching event listeners to multiple elements, you can add a listener to a parent element and use event. target to identify which child was interacted with that is event Delegation.

What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and propagates upwards through its ancestors, triggering their event listeners.

How do you create, add, and remove elements using JavaScript?

  • Create: Use document.createElement("p").
  • Add: Use appendChild(), prepend(), insertBefore(), or innerHTML.
  • Remove: Use removeChild() or remove().
